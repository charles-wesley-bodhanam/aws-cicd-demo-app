\# AWS CI/CD Demo App



This is a simple Node.js application used to demonstrate a CI/CD pipeline on AWS using:



\- AWS CodePipeline

\- AWS CodeBuild

\- Amazon ECR

\- Amazon EC2

\- Docker



\## Endpoints



\- `/` – Basic home page

\- `/health` – Health check endpoint (used for monitoring \& deployment checks)



\## Run locally



```bash

npm install

npm start
